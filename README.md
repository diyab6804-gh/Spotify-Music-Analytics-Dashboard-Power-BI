# Spotify Music Analytics Dashboard | Power BI

## Project Overview
This project analyzes Spotify music data using Power BI to uncover insights related to song popularity, artist performance, album types, and content trends.

---

## Objectives
- Analyze Spotify songs and artists using key metrics  
- Identify popularity trends across years and quarters  
- Compare explicit vs non-explicit content  
- Understand album type performance (Single vs Album)  

---

## Tools & Technologies
- Power BI Desktop  
- Power Query  
- DAX  

---

## Key Metrics
- Total Songs  
- Total Artists  
- Explicit Songs Count  
- Average Song Duration  
- Average Popularity  

---

## Dashboard Features
- Songs by Artist analysis  
- Album type comparison  
- Explicit vs Non-explicit distribution  
- Year-wise and quarter-wise popularity trends  
- Interactive artist filter  

---

## Skills Demonstrated
- Data Cleaning & Transformation  
- Data Visualization  
- DAX Calculations  
- Analytical Thinking  

---

## How to Use
1. Download the `.pbix` file  
2. Open it in Power BI Desktop  
3. Explore insights using filters and visuals  

---

## Conclusion
This dashboard demonstrates the use of Power BI to convert raw music data into meaningful insights through interactive visualizations.
